IQ Intelligent School Solution

Introduction:

Composed of IQSchool Information Distribution System, IQSchool Digital Class Card System, IQ Centralized Management System and IQSchool One-card System, IQ Intelligent School Solution can comprehensively manage information and safety of school.

Component:
Feature:

Automatically display information including current and next classes, subjects and teachers according to course timetable.

Automatically display students' attendances by support of IQSchool One-card System.

A variety of information, such as class presence, class presentation, video of class meetings and so on, can be displayed on each terminal instead of traditional blackboard.

Multi-user management pattern. Different administration rights can be set. Integrated with functions including firewall, data encryption, terminal authentication, log auditing, regular backup management.

Statistically analyze the usage of all LEDs. Generate graphic report of items such as operating rate, startup time and active degrees rank.

The system integrated with functions including monitoring screens of any LED in real time, viewing running state and remotely maintaining software issues, which greatly simplifies management for devices.

The system can uniformly manage all LEDs by group, such as real-time information release, remote command, remote startup& shutdown, signals switch and volume adjustment, which greatly improves the efficiency of management.
